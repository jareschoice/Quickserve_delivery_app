# QuickServe Backend

Run:
```
npm install
node server.js
```

Env keys (create `.env`):
```
PORT=5000
MONGO_URI=...
EMAIL_HOST=mail.getquickserves.com
EMAIL_PORT=465
EMAIL_SECURE=true
EMAIL_USER=admin@getquickserves.com
EMAIL_PASS=***
EMAIL_FROM=admin@getquickserves.com
PAYSTACK_PUBLIC_KEY=...
PAYSTACK_SECRET_KEY=...
PAYSTACK_CURRENCY=NGN
COMMISSION_PLATFORM=50
COMMISSION_VENDOR=50
COMMISSION_RIDER=50
JWT_SECRET=supersecret_quickserve
JWT_EXPIRES=7d
APP_BASE_URL=http://localhost:5000
```
